Radio Edit

A tool to launch a nova, glance, dashboard and keystone environment inside rackspace cloud. It does so by downloading the latest version of everything (running auto.sh on boot). It takes around 5 minutes and if successful have an up arrow next to the name.

Using stacks

  • Dashboard: Click link (name) to view the dashboard and play around.
  • Broken? ssh as root and use screen to see what's up hint: screen -x
  • check the install log at /var/log/install.log

Ideas for improvements? Click fork on github and then send a pull request with your patch.
